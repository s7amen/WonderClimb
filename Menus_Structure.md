# Меню структура

Primary menu (horizontal) - for public users
    - Начало
    - График
    - Календар


Secondary menu (horizontal) - for logged in users with role climbers
    - Табло
    - Моя график
    - График
    - Абонаменти
    - Профил

Vertical primary menu + vertical secondary menu (vertical) - for logged in users with role more that climbers
    - Зала (role instructor+ (instructor, coach, admin))
        - Табло
        - Карти (gym pass)
        - Продукти
        - Посещения (gym visits)
        - Катерачи (включително и секция Семейства)
        - Цени (instructor and coach can view, admin can edit)
        

    - Тренировки (coach and admin)
        - Табло ( за сега е празно) - coach dashboard
        - Тренировки - следващите тренировки от граафик със записаните катерачи (направените резервации).(Това е мястото, където:

„За всяка тренировка се виждат резервациите, може да се добавят/трият резервации, да се потвърждава присъствие, да идват деца без booking“) Тук са и всички опции за провеждане на тренировки
        - График (с опции за редактиране и добавяне на тренировки)
        - Резервации (booking)
        - Катерачи (същата страница като мено катерачи в Зала)
        - Треньори
        - Състезания (с опция за редактиране)
        - Карти (training pass)
        - Цени (instructor and coach can view, admin can edit) - същата страница като в Зала но с филтър за цените на тренировките само
        - Плащания (Coach Payouts) - admin only
    - Финанси (admin)
        - Финансов дневник (всички приходи и разходи) - FinanceEntry
        - Финанси зала - Gym Finance
        - Финанси тренировки - Training Finance
    - Настройки (admin)
        - Настройки на съобщения
        - Notifications
        - System Settings
